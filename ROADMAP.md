# Roadmap 🗺️ FocusUp Web

Este documento muestra la dirección del proyecto y el orden sugerido de evolución. 
La comunidad puede proponer cambios en este orden mediante un issue o PR.

## ✅ Fase 0 — MVP (completado)
- [x] Temporizador Pomodoro básico (25 min)
- [x] Botones iniciar / pausar / resetear

## 🔜 Fase 1 — Persistencia básica
- [ ] Guardar sesiones completadas en localStorage
- [ ] Mostrar contador de sesiones en pantalla

## 🔜 Fase 2 — Ciclo Pomodoro completo
- [ ] Descanso automático (5 min) tras cada sesión
- [ ] Descanso largo (15-20 min) cada 4 sesiones
- [ ] Configuración de duración personalizada

## 🔜 Fase 3 — Panel de progreso
- [ ] Vista semanal de sesiones completadas
- [ ] Gráfico simple (barras o círculos)

## 🔜 Fase 4 — Bloqueo de distracciones
- [ ] Extensión de navegador básica
- [ ] Lista configurable de sitios a bloquear

## 🔜 Fase 5 — Colaboración
- [ ] Salas de estudio con contador compartido (WebSockets / Firebase)
- [ ] Indicador de cuántos usuarios están en la sala

## 💡 Ideas a futuro (sin fecha)
- Modo oscuro
- Integración con Google Classroom / Moodle
- Estadísticas de horas más productivas
