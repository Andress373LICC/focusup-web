# Guía para colaboradores 🤝

¡Gracias por tu interés en contribuir a FocusUp Web!

## Cómo empezar
1. Haz un fork del repositorio.
2. Crea una rama para tu mejora (`git checkout -b mi-mejora`).
3. Realiza tus cambios y haz commit.
4. Envía un pull request explicando tu aporte.

## Ideas de contribución
- Mejorar el temporizador (descansos automáticos, configuraciones).
- Diseñar interfaz más atractiva y minimalista.
- Implementar panel de progreso con localStorage.
- Crear extensión de navegador para bloqueo de distracciones.
- Añadir salas colaborativas con contador compartido.

## Estilo de código
- Usa HTML, CSS y JavaScript simples.
- Mantén la interfaz limpia y minimalista.
- Documenta tus funciones con comentarios claros.
